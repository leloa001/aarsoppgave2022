*For å kjøre programmet* 

1. Last ned alle .java filene fra GitHub lenken
2. Åpne XAMPP server med riktig navn og IP (ip: 10.0.0.86, eller du kan endre koden i getConnection() under gamePannel klassen)
3. Lage en DB som heter snakeGameDB med en tabel som heter scoretabel med parameter(UserName, Score)
4. Sjekke at klienten og server når hverandre (hvis ikke åpne brannmur)
5. Kjøre snakeGame.java og sjekke at alt virker
6. Om ingenting virker be til gud
